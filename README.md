# Simple Simon Says Game Built on JavaScript

This project is a simple implementation of the classic Simon Says memory game built using JavaScript. The game challenges players to remember and repeat a sequence of colors that increases in length with each level.

It is designed to demonstrate core JavaScript concepts such as event handling, DOM manipulation, game logic, and timing functions while creating a fun and interactive browser-based game.

# How the Game Works
	1.	The game begins when the user presses a key or clicks the start button.
	2.	The system randomly selects a color and flashes it.
	3.	The player must click the same color.
	4.	With each level, the sequence becomes longer.
	5.	If the player clicks the wrong color, the game ends and displays the final score.

# Features
	•	Interactive gameplay
	•	Increasing difficulty with levels
	•	Random color sequence generation
	•	User input detection
	•	Game over and restart functionality
	•	Visual feedback using color flashes and animations
    •   Highscore display using localstorage
